轻量级GULP WEB框架（Simplify-gulp）  
 A lightweight WEB framework (Simplify-gulp)
====


## 概述

1. 实现CSS合并压缩
2. 实现JS合并压缩
3. 实现文件与文件夹监控、同步
4. 实现非压缩CSS、JS调试
5. 实现服务器端口修改
6. 浏览器前缀和兼容等问题
7. 给压缩的CSS和JS加头注释
8. 支持less
